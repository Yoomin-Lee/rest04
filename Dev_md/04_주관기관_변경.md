# 04. 주관 기관을 김예진 동물병원으로 변경

작업일: 2026-06-09  
커밋: `43fabec`

---

## 목표
PawEdu 교육 플랫폼의 주관 기관을 **김예진 동물병원**으로 설정한다.  
플랫폼 브랜드명(PawEdu)은 유지하고, 운영 주체가 김예진 동물병원임을 전 사이트에 반영한다.

---

## 변경 파일

### `src/data/site.js`
| 필드 | 변경 전 | 변경 후 |
|------|---------|---------|
| `fullName` | `PawEdu Co., Ltd.` | `김예진 동물병원` |
| `copyright` | `© 2026 PawEdu Co., Ltd.` | `© 2026 김예진 동물병원` |
| `taglineSub` | AI와 수의학 전문 지식이 만나는 반려동물 교육 플랫폼 | **김예진 동물병원**이 직접 운영하는 반려동물 전문 교육 플랫폼 |
| `intro[0]` | 포에듀는 반려동물의 건강한 삶을 위한… | **김예진 동물병원**이 직접 만들고 운영하는… |
| `offices[0].label` | `본사` | `본원` |

### `src/pages/About.jsx` — CEO 인사말
| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 이름 | 홍길동 | **김예진** |
| 직함 | CEO / 대표이사 | **원장 / 수의사** |
| 인사말 내용 | PawEdu 창업자 이야기 | **수의사 임상 경험** 기반으로 PawEdu를 시작하게 된 배경 |
| 서명 | PawEdu Co., Ltd. 대표이사 | 김예진 동물병원 원장 |

---

## 영향 범위
`site.js`를 참조하는 모든 컴포넌트에 자동 반영된다.

- `Footer` — `fullName`, `copyright`, `offices` 주소·연락처
- `Privacy` — 회사명, 이메일, 전화번호
- `Terms` — 회사명, 이메일, 전화번호
- `Contact` — 주소, 전화, 이메일
- `About` — CEO 섹션 (별도 수동 수정)

---

## 검증
- `npm run build` 성공
- 푸터 카피라이트 `© 2026 김예진 동물병원` 확인
- CEO 인사말 이름·직함·내용 정상 표시 확인
- 개인정보처리방침·이용약관 회사명 자동 반영 확인
