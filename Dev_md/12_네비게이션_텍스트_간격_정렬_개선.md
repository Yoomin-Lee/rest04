# 12. 네비게이션 바 텍스트 간격 조정 — 드롭다운 정렬 개선

작업일: 2026-06-11

---

## 목표
드롭다운 메뉴 간격에 맞게 네비게이션 바 텍스트 간격 조정

---

## 문제

이전 작업(#11)에서 드롭다운 컨테이너를 `left-0`으로 변경한 후, 네비 텍스트와 드롭다운 텍스트의 시작점이 다른 문제가 남아 있었음.

| 요소 | 적용 값 | 텍스트 시작 x 위치 |
|------|---------|-----------------|
| 네비 링크 | `px-5` (20px 양쪽) | `<li>.left + 20px` |
| 드롭다운 컨테이너 | `left-0` | `<li>.left + 0` |
| 드롭다운 아이템 | `px-5` (20px) | `<li>.left + 20px` |

수정 시도 1차: 네비 링크를 `pl-0 pr-5`로 변경 → 오히려 **비대칭** 발생
- 네비 텍스트: `<li>.left + 0`
- 드롭다운 텍스트: `<li>.left + 16px` (아이템 px-4 기준)
- 텍스트 시작점 불일치로 더 어색해짐

---

## 해결

네비 링크와 드롭다운 아이템 모두 **`px-4`(16px)** 로 통일.

```
네비 링크 텍스트:   <li>.left + 16px
드롭다운 텍스트:    <li>.left + 0 (left-0) + 16px (px-4) = <li>.left + 16px
```

→ 두 텍스트 시작점이 동일 ✓

### 변경 내용

```jsx
// 네비 링크 패딩
// 이전: 'px-5 py-5 text-[15px] font-bold transition'
// 이후: 'px-4 py-5 text-[15px] font-bold transition'

// active 밑줄
// 이전: 'absolute bottom-0 left-5 right-5 h-0.5 rounded-full'
// 이후: 'absolute bottom-0 left-4 right-4 h-0.5 rounded-full'

// 드롭다운 아이템
// 이전: 'block whitespace-nowrap px-5 py-2.5 text-sm ...'
// 이후: 'block whitespace-nowrap px-4 py-2.5 text-sm ...'
```
