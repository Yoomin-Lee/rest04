# 03. 미완성 페이지 완성 (강사·파트너십·개인정보처리방침·이용약관)

작업일: 2026-06-09  
커밋: `7307297`

---

## 목표
`SimplePage`(준비 중 placeholder)로 처리되어 있던 4개 라우트에 실제 콘텐츠 페이지를 구현한다.

---

## 구현 페이지

### `/instructors` → `Instructors.jsx`
- `site.js`의 `instructors` 배열을 읽어 카드 그리드로 표시
- 강사별 전문 분야 색상 배지(support·primary·accent) 적용
- 카드에 강의 수·평점 통계 표시
- 하단에 파트너십 페이지로 연결되는 CTA 배너

### `/partnership` → `Partnership.jsx`
| 섹션 | 내용 |
|------|------|
| 통계 | 수강생 1,200+ / 콘텐츠 100+ / 만족도 4.9 / iOS·AOS |
| 파트너 유형 | 동물병원·AI기업·교육기관·반려동물 브랜드 (4종 카드) |
| 진행 절차 | 문의 접수 → 사전 미팅 → 계약 체결 → 콘텐츠 협력 (4단계) |
| CTA | `/contact/partnership`으로 연결 |

- 시행일: 2026-04-01

| 제7조 | 개인정보 파기 |
