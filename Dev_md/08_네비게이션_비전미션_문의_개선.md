## 2. 네비게이션 바 전면 개선

### 버그 수정

- **"교육 콘텐츠"** NavLink `to="/videos/ai"` → 다른 영상 카테고리 (`/videos/literacy` 등) 접속 시 active 표시 안 되던 문제 수정
- **"강사 소개"** — `/partnership` 접속 시에도 active 표시되도록 자식 경로 포함 판별 추가

```js
function isNavActive(item, pathname) {
  if (pathname.startsWith(item.to.split('/').slice(0, 2).join('/'))) return true
  return item.children?.some(c => pathname.startsWith(c.to)) ?? false
}
```

### 가시성 개선

- active 항목 하단 `h-0.5` 밑줄 인디케이터 추가 (solid 헤더: `bg-primary-600`, 투명 헤더: `bg-white`)
- 폰트 `font-semibold` → `font-bold`
- 모바일 메뉴: 현재 섹션 항목 `text-primary-600` 강조

### 드롭다운 위치 정렬 수정

- **기존**: 헤더 하단 별도 분리된 mega-dropdown에 `w-40` 고정 컬럼 → nav 항목 위치와 불일치
- **변경**: 각 `<li>`에 `absolute left-1/2 -translate-x-1/2` 드롭다운 직접 부착
- `onMouseLeave`를 `<li>` 단위 처리 → 항목 ↔ 드롭다운 간 이동 시 깜빡임 없음
- `whitespace-nowrap`으로 드롭다운 내 텍스트 줄바꿈 방지

## 3. 비전/미션 페이지 구체화

### 추가 섹션

| 섹션 | 내용 |
|------|------|
| Vision 카드 | 제목 + 본문 + 체크리스트 3항목 |
| Mission 카드 | 제목 + 본문 + 체크리스트 3항목 |

| Strategic Pillars | 콘텐츠 전문성 · AI 개인화 · 커뮤니티 연결 · 사회적 가치 (4항목) |
| Core Values | 신뢰 · 혁신 · 공감 · 접근성 — 단순 태그라인 → 실천 내용 서술 |

### 가독성 개선 (2회 수정)

**문제**: 비전·미션·Strategic Pillars 카드 모두 색 계열 배경(`bg-support-50`, `bg-primary-50` 등)에 동일 계열 텍스트(`text-support-700`, `text-primary-700`) 사용 → 명도 대비 부족

**해결**:
- 카드 배경: tinted bg → `bg-white dark:bg-deep-800` (중립, `card-base`)
- 아이콘: 컬러 배지 블록 (`bg-*-100 dark:bg-*-900/50`) 으로 색 포인트 유지
- 체크리스트/설명 텍스트: `text-neutral-700 dark:text-neutral-300` 통일

## 4. 제휴 문의 이메일 노출

`Contact.jsx` — 제휴 문의 탭 폼 상단에 이메일 배너 추가

```jsx
{type === 'partnership' && (
  <div className="flex items-center gap-4 rounded-xl border border-primary-200 bg-primary-50 ...">
    <Icon name="mail" ... />
    <a href={`mailto:${company.offices[0].email}`}>
      {company.offices[0].email}
    </a>
  </div>
)}
```

- 클릭 시 `mailto:` 로 메일 앱 직접 연결
