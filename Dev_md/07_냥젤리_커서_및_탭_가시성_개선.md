# 07. 냥젤리 커스텀 커서 및 탭 가시성 개선

작업일: 2026-06-09

---

## 목표

1. 마우스 커서를 브랜드 색상의 냥젤리(고양이 젤리 발바닥) 이미지로 교체
2. 서브 페이지 탭 버튼의 크기와 색 대비를 높여 가시성 개선

---

## 변경 내용

### 신규 파일: `public/cursor.svg`

- 40×40px SVG 커스텀 커서
- 고양이 발바닥(냥젤리) 디자인: 발가락 젤리 4개 + 메인 패드
- 색상 팔레트:
  - 발가락 젤리: `#c4b5fd` (primary-300, 연보라)
  - 메인 패드: `#7c3aed` (primary-600, 브랜드 보라)
  - 하이라이트: 흰색 반투명 (젤리 질감 표현)
- 흰 후광 레이어로 어두운 배경에서도 가시성 확보
- CSS 핫스팟: `21 28` (메인 패드 중심)

### 수정 파일: `src/index.css`

#### 냥젤리 커서 적용

```css
* {
  cursor: url('/rest04/cursor.svg') 21 28, auto;
}
input[type="text"],
input[type="email"],
input[type="search"],
textarea {
  cursor: url('/rest04/cursor.svg') 21 28, text;
}
```

- 전체 요소(`*`)에 커스텀 커서 적용
- 텍스트 입력 요소는 text 폴백 유지

#### 탭 버튼 가시성 개선

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 패딩 | `px-5 py-2.5` | `px-6 py-3` |
| 폰트 크기 | `text-sm` | `text-[15px]` |
| 폰트 굵기 | `font-semibold` | `font-bold` |
| 테두리 형태 | `rounded-full` | `rounded-xl` |
| 비활성 배경 | `bg-neutral-100` | `bg-white` + `border-2 border-neutral-200` |
| 활성 그림자 | `shadow-sm` | `shadow-md shadow-primary-600/30` |
| 비활성 호버 | `hover:bg-primary-100` | `hover:border-primary-400 hover:bg-primary-50` |
| 다크 비활성 | `dark:text-primary-300` | `dark:text-primary-200` (더 밝음) |

- 비활성 탭에 테두리 추가 → 탭 간 구분 명확화
- 활성 탭 그림자 강화 → 선택 상태 강조
- 다크모드 비활성 탭 텍스트 밝기 향상
