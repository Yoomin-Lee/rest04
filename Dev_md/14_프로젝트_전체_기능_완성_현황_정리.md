## 기술 스택

| 분류 | 기술 |
|------|------|
| 프레임워크 | React 18 + Vite 5 |

| `/register` | `Register` | 이메일 회원가입 |

## 컴포넌트 현황

| 컴포넌트 | 역할 |
|----------|------|
| `Header` | 네비게이션 바 — 드롭다운·다크모드 토글·반응형 |

| `ChatWidget` | 오른쪽 하단 AI 채팅 팝업 (Solar / OpenAI 탭 선택) |
| `ScrollToTop` | 라우트 변경 시 페이지 상단 이동 |
| `ScrollToTopButton` | 스크롤 위치에 따라 표시되는 상단 이동 버튼 |
| `ThemeToggle` | 다크/라이트 모드 전환 버튼 |
| `VideoCard` | 영상 썸네일 카드 |
| `VideoModal` | YouTube 영상 팝업 플레이어 |
| `SubPageLayout` | 서브 페이지 공통 레이아웃 래퍼 |
| `Icon` | SVG 아이콘 통합 컴포넌트 |
| `Placeholder` | 이미지 로딩 전 대체 UI |

## Supabase 스키마 (테이블)

모든 테이블은 CatsCare 앱과의 충돌 방지를 위해 `pawedu_` 접두어 사용.

| 테이블 | 내용 |
|--------|------|
| `pawedu_posts` | 게시글 (제목·내용·작성자·카테고리·조회수) |
| `pawedu_comments` | 댓글 (게시글 FK·작성자·내용) |
| `pawedu_post_files` | 게시글 첨부파일 (Storage 버킷 경로 참조) |

## Edge Function

| 함수 | 경로 | 역할 |
|------|------|------|
| `chat` | `supabase/functions/chat/index.ts` | Solar API → OpenAI 순으로 AI 응답 프록시 |

- 배포: `npx supabase@latest functions deploy chat --no-verify-jwt --project-ref kbjxjogmnwurxbxnpfsz`
- 시크릿: `SOLAR_API_KEY`, `OPENAI_API_KEY` (Supabase Dashboard → Settings → Secrets)

## UX 기능

| 기능 | 구현 방식 |
|------|----------|
| 다크모드 | `ThemeContext` + `localStorage` 영속화 |
| 냥젤리 커서 | CSS `cursor: url(...)` — 탭·버튼 호버 시 보라색 전환 |
| 스크롤 상단 이동 | `ScrollToTopButton` (화면 하단 고정) |
| 반응형 | Tailwind 브레이크포인트 (`sm:` `md:` `lg:`) |
